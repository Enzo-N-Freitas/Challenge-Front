Sobre a Solução

O que é o avatar inteligente proposto?
É um assistente virtual integrado à plataforma SoulUp, capaz de interagir com o usuário de forma contextual e personalizada — sugerindo ações sustentáveis, acompanhando missões e criando publicações automaticamente com base no perfil de cada usuário.
Como o avatar se adapta ao perfil do usuário?
O avatar analisa o histórico de ações e interações do usuário para ajustar seu comportamento dinamicamente. Inicialmente com regras simples, a solução evolui progressivamente para modelos de personalização mais avançados.
Quais tecnologias foram utilizadas?
O front-end foi desenvolvido com React, HTML e CSS. Para o back-end e automações, foram utilizadas outras ferramentas e linguagens complementares, com banco de dados escolhido e justificado pelo grupo de acordo com a proposta.
Como a privacidade do usuário é protegida?
A solução adota análise baseada em padrão e recorrência, reduzindo falsos positivos e garantindo que dados sensíveis sejam tratados com proteção adequada. Há também mecanismos de revisão e escalonamento em situações críticas.
A solução é escalável para muitos usuários?
Sim. A arquitetura proposta leva em conta a escalabilidade para múltiplos usuários simultâneos e a performance na geração de recomendações em tempo real, sendo um dos principais critérios de avaliação da proposta.
O layout do projeto é responsivo?
Sim. Todas as páginas foram desenvolvidas com CSS responsivo, adaptando-se corretamente a desktop, tablet e dispositivos móveis.
