# Challenge Front-End

## 📖 Sobre o Projeto

O **Challenge Front-End** é um projeto acadêmico desenvolvido com foco em design, responsividade e experiência do usuário. O sistema apresenta uma interface moderna utilizando HTML, CSS e JavaScript.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Git & GitHub
* Claude AI (suporte visual e estilização)

---

## 📂 Estrutura do Projeto

```bash
Challenge-Front/
├── css/
├── img/
├── paginas/
├── index.html
└── README.md
```

---

## 🎨 Funcionalidades

* Interface responsiva
* Navegação entre páginas
* Efeitos visuais com blur
* Organização visual com cards
